# NOIR Fashion Website

A minimalist fashion website for the NOIR brand.

## Description

This is a static website for NOIR, a fashion brand specializing in minimalist, sustainable luxury clothing. The site showcases the SS 2026 Collection with a focus on "Dress the Silence" theme.

## Features

- Responsive navigation bar with links to all pages
- Hero section featuring the latest collection
- Animated marquee with announcements
- Image strips highlighting featured drops, new arrivals, and journal entries
- Footer with quick links and copyright information

## Pages

- `index.html`: Home page with hero, marquee, and image strips
- `about.html`: About the brand
- `services.html`: Services offered
- `portfolio.html`: Portfolio of work
- `blog.html`: Journal/Blog
- `contact.html`: Contact information
- `faq.html`: Frequently Asked Questions

## Technologies Used

- HTML5
- CSS3 (custom styles in `shared.css`)
- Bootstrap 5.3.2
- Font Awesome 6.5.0
- Google Fonts (Cormorant Garamond and Montserrat)

## Installation

1. Clone or download the project files.
2. Ensure all dependencies (Bootstrap, Font Awesome, Google Fonts) are accessible via CDN.
3. Open `index.html` in a web browser.

## Usage

Navigate the website using the navbar. Links open in new tabs for better user experience.
